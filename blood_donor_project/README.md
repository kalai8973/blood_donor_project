# 🩸 Blood Donor Finder

A Django-based Blood Donor Finder web application.

## Features

- Register Blood Donors
- Search Donors by Blood Group & City
- Blood Request Module
- Dashboard Analytics
- Charts (Chart.js)
- PDF Export
- Excel Export
- Email Notifications
- User Authentication
- Responsive Bootstrap UI

## Technologies Used

- Python
- Django
- SQLite
- Bootstrap 5
- Chart.js
- ReportLab
- OpenPyXL
- DataTables

## Installation

```bash
git clone https://github.com/kalai8973/blood_donor_project.git

cd blood_donor_project

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
